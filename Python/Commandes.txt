Ce fichier est un récapitulatif des commandes utilisables au clavier dans Pokémon Origins.

Attention : il est présent uniquement à titre informatif, modifier ce fichier ne modifiera pas les commandes du jeu !

N'importe quand en jeu : 

— Alt+Entrée(sous windows) ou Alt+cmd(pomme)+Entrée(sous Mac): Mettre en plein écran.
— Alt + F4 : Fermer le jeu.
— F1 : Modifier les commandes, options diverses.
— F12 : Redémarrer le jeu.

Commandes du personnage :

— Touches directionnelles : Déplacer.
— Entrée/Barre d'espace/C : Interagir.
— Echap/X/0 (maintenir) : Ouvrir le menu.
— S/D : Utiliser un objet en raccourci.
— Q : Augmenter la vitesse de la bicyclette.
— W : Réduire la vitesse de la bicyclette.

Pendant un dialogue :

— Entrée/Barre d'espace/C : Passer, valider.
— Echap/X/0 (maintenir) : Passer plus rapidement.

Dans n'importe quelle interface (sac, Pokédex, etc...) :

— Touches directionnelles : Déplacer le curseur.
— Entrée/Barre d'espace/C : Valider.
— Echap/X/0 : Annuler.
— Q : Monter rapidement le curseur.
— W : Descendre rapidement le curseur.

Dans le menu Echap et le menu Equipe :

— Clic gauche : Sélectionner un Pokémon ou une option.
— Clic droit (maintenir) : Déplacer un Pokémon.

Dans le sac :

— S/D : Assigner un objet à la touche S ou D.
— A : Trier les objets par ordre alphabétique / par numéro ID.

Dans le Pokédex : 

— Shift/A : Cacher les entrées vides.

Dans le PC de stockage :

— A : Positionner le curseur sur l'équipe.
— A (dans le menu de recherche) : Positionner le curseur sur "Lancer la recherche".
— S : Positionner le curseur sur la boîte.
— D : Positionner le curseur sur le nom de la boîte.
— Q/W : Se déplacer d'une boîte à l'autre.

Dans l'interface d'écriture d'un surnom :

— Echap/X/0 : Effacer un caractère.
— A : Placer le curseur sur OK.
